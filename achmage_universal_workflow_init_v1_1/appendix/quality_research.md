# appendix/quality_research.md — Quality Gates for Research (v1.1)

목표: 연구/분석 산출물의 논리·재현성·근거를 안정적으로 확보하기.

## A) Quick Gate (5분)
- [ ] 연구 질문/목표가 한 문장으로 명확하다.
- [ ] 데이터/자료의 출처가 SOURCES에 기록되어 있다.
- [ ] 방법(Methods)이 “누가 읽어도 따라할 수 있게” 적혀 있다.
- [ ] 결과(Results)와 해석(Discussion)을 분리했다.
- [ ] 한계(Limitations) 2개 이상을 명시했다.

## B) Deep Gate (15~30분)
- [ ] 변수/개념 정의가 일관되고 측정 가능하다.
- [ ] 분석 절차가 단계별로 명시되어 있다(전처리→분석→검증).
- [ ] 대안 가설/반례 가능성을 논의했다.
- [ ] 표/그림이 주장과 직접 연결된다(장식용 X).
- [ ] 인용/참고문헌 형식이 일관된다(APA 등).

## C) 재현성 체크(가능할 때)
- [ ] 원자료 또는 요약 데이터가 data/에 있다(또는 접근 방법이 기록되어 있다).
- [ ] 분석 코드/절차가 있다(없으면 “왜 없는지”와 대안 기록).
- [ ] 랜덤 시드/버전/환경이 기록되어 있다(가능한 범위에서).

## D) 권장 프롬프트(검토용)
- “이 연구 초안을 RQ/Methods/Results/Discussion/Limitations 관점에서 심사하고, 빠진 항목을 체크리스트로 표시해줘.”
