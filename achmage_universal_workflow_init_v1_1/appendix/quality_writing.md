# appendix/quality_writing.md — Quality Gates for Writing (v1.1)

목표: 글 산출물의 일관성과 완성도를 안정적으로 올리기.
방식: Quick Gate(빠른 점검) → Deep Gate(심층 점검)

## A) Quick Gate (3분)
- [ ] 목표(Goal) 한 문장으로 요약 가능하다.
- [ ] 독자/대상(audience)이 명확하다.
- [ ] 제약(금지/필수/분량)이 지켜졌다.
- [ ] 구조(OUTLINE)와 본문이 대응된다.
- [ ] “다음에 무엇을 고치면 더 좋아지는지” 3개가 즉시 나온다.

**Fail 시 Fix**
- “OUTLINE과 본문 불일치” → OUTLINE을 먼저 업데이트하거나, 본문을 OUTLINE에 맞게 재배치
- “톤/대상 불명확” → BRIEF에 대상/톤 2문장 추가 후 재작성

## B) Deep Gate (10~20분)
- [ ] 도입부에 훅(문제/긴장/질문)이 있다.
- [ ] 중간부는 단순 반복이 아니라 진행(발전)이 있다.
- [ ] 결말은 독자가 가져갈 메시지/정서가 남는다.
- [ ] 문장 수준: 불필요한 수식어/중복 표현을 제거했다.
- [ ] 사실/참조가 필요한 부분은 SOURCES에 근거가 있다(또는 “추정” 표기).

## C) 산출물 유형별 체크
### 소설/서사
- [ ] 인물의 목표/장애/대가가 명확하다.
- [ ] 장면 전환마다 목적이 있다(정보/감정/사건).
### 에세이/칼럼
- [ ] 주장(Thesis) 1개가 끝까지 유지된다.
- [ ] 반례/다른 관점 1개 이상을 다뤘다.
### 강의노트/설명문
- [ ] 예시(Example)가 최소 2개 들어간다.
- [ ] 용어 정의가 일관된다.

## D) 권장 프롬프트(검토용)
- “이 글을 OUTLINE과 BRIEF 기준으로 검사하고, 실패 항목과 수정안을 제시해줘.”
